# Supermarket Sales Dashboard | Power BI

## Problem Statement
Analyze supermarket sales data to track performance, identify trends, and support business decisions.

## Tools Used
- Power BI
- Power Query
- Basic DAX

## Dashboard Features
- Total Sales, Revenue, and Key KPIs
- Sales trends over time
- Product category performance
- Region-wise analysis

## Steps Performed
- Cleaned and transformed raw data using Power Query
- Created calculated measures using DAX
- Built interactive dashboard with filters and visuals

## Key Insights
- Identified top-performing product categories
- Observed sales trends across different time periods
- Highlighted high and low-performing regions
